Low-power 64-bit ALU using Verilog HDL and Vivado with operand isolation, clock-enable control, and CLA techniques achieving up to 99.56% power reduction on FPGA platforms.

# 64-bit ALU Power Optimization

## Project Overview
This project presents the design and implementation of a low-power 64-bit ALU using optimization techniques such as:

- Operand Isolation
- Clock Enable Based Activity Control
- Carry Look Ahead Arithmetic

Implemented using Verilog HDL in Xilinx Vivado.

## Objectives
- Reduce dynamic power consumption
- Minimize switching activity
- Compare normal and optimized ALU architectures

## Tools Used
- Xilinx Vivado
- Verilog HDL
- FPGA Boards

## Results
Power reduction achieved up to 99.56%.

## FPGA Boards Tested
- Zynq 7000
- Spartan-7
- Artix-7
- Kintex Ultrascale
- Kintex-7
